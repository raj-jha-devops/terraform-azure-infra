# Terraform Azure Infrastructure

## Overview
This project demonstrates how to provision Azure infrastructure using Terraform
following Infrastructure as Code (IaC) best practices.

## Features
- Azure Resource Group provisioning
- Modular Terraform structure
- Provider and variable abstraction
- Ready for CI/CD integration

## Architecture
Azure Subscription  
└── Resource Group

## Technologies Used
- Terraform
- Microsoft Azure
- IaC Best Practices

## How to Use
```bash
terraform init
terraform plan
terraform apply
